# RadioTrack - ระบบยืม-คืน วิทยุสื่อสาร

ระบบจัดการการยืม-คืนวิทยุสื่อสารแบบออนไลน์ พร้อมฟีเจอร์ครบครัน

## 🌟 ฟีเจอร์หลัก

- ✅ **การยืม-คืนวิทยุ** จัดการการยืมและคืนวิทยุสื่อสารได้ 84 เครื่อง
- 📷 **ถ่ายรูป/อัปโหลดหลักฐาน** ถ่ายรูปผู้ยืมผ่านกล้อง หรืออัปโหลดรูปภาพจากเครื่อง
- 🔍 **ค้นหาอัจฉริยะ (Autocomplete)** เชื่อมข้อมูลชื่อผู้ยืมและหมายเลขวิทยุจากไฟล์ Excel อัตโนมัติ
- 📱 **สแกน QR Code** สแกน QR Code เพื่อเลือกวิทยุได้รวดเร็ว
- 📊 **รายงานสรุป** แสดงสถิติวิทยุที่ว่างและถูกยืม
- 📄 **พิมพ์เอกสาร** พิมพ์ใบยืม-คืน (แบบ ทอ.๓๕) ได้ทันทีในขนาด A4
- 📈 **Export CSV** ส่งออกข้อมูลเป็นไฟล์ CSV
- 🔐 **ระบบล็อกอิน** ควบคุมสิทธิ์การเข้าใช้งาน
- ☁️ **Google Sheets Sync** ซิงค์ข้อมูลอัตโนมัติผ่านไฟล์ config.js

## 📁 โครงสร้างไฟล์

```
commu1/
├── css/
│   └── styles.css          # ไฟล์สไตล์ (UI/UX)
├── js/
│   ├── app.js              # Logic หลักของระบบ
│   ├── config.js           # ไฟล์ตั้งค่า Google Sheets
│   ├── radios.js           # ข้อมูลวิทยุ (จาก hytera.xlsx)
│   ├── users.js            # ข้อมูลผู้ใช้งาน (จาก nn.xlsx)
│   └── google-sheets-api.js # โมดูลเชื่อมต่อ Google Sheets
├── excel/
│   ├── nn.xlsx             # ไฟล์รายชื่อต้นฉบับ
│   └── hytera.xlsx         # ไฟล์ข้อมูลวิทยุต้นฉบับ
├── index.html              # หน้าเว็บหลัก
├── evidence-form.html      # หน้าสำหรับพิมพ์ใบเบิก (A4)
├── package.json            # ไฟล์สำหรับรันด้วย npm
├── .gitignore              # ไฟล์ยกเว้นไฟล์ชั่วคราว
└── README.md               # คู่มือการใช้งาน
```

## 🚀 การติดตั้งและใช้งาน

### 1. ดาวน์โหลดไฟล์
ดาวน์โหลดไฟล์ทั้งหมดในโฟลเดอร์ `commu1/`

### 2. เปิดระบบ
เปิดไฟล์ `index.html` ในเว็บเบราว์เซอร์ (Chrome, Firefox, Edge)

### 3. เข้าสู่ระบบ
- ใส่อีเมลที่ได้รับอนุญาต (default: nuttapop14@gmail.com)
- คลิก "เข้าสู่ระบบ"

### 4. เริ่มใช้งาน
- **แถบซ้าย**: ฟอร์มยืมวิทยุและคืนวิทยุ
- **แถบขวา**: ตารางรายการทั้งหมดและสถิติ

## 🔧 การตั้งค่า Google Sheets (Optional)

หากต้องการแก้ไขการเชื่อมต่อ Google Sheets:

1. เปิดไฟล์ `config.js`
2. แก้ไขข้อมูล `spreadsheetId` และ `apiKey`
3. บันทึกและรีเฟรชหน้าเว็บ ระบบจะซิงค์ข้อมูลให้อัตโนมัติ

## 👥 การจัดการผู้ใช้

### เพิ่มผู้ใช้ที่ได้รับอนุญาต
แก้ไขไฟล์ `app.js` ในส่วน:

```javascript
this.AUTHORIZED_EMAILS = [
  'nuttapop14@gmail.com',
  'user1@company.com',    // เพิ่มอีเมลที่นี่
  'user2@company.com'     // เพิ่มอีเมลที่นี่
];
```

## 📱 การใช้งานบนมือถือ

ระบบรองรับการใช้งานบนมือถือ:
- ใช้กล้องถ่ายรูปผู้ยืม
- สแกน QR Code ผ่านกล้องหลัก
- Responsive design สำหรับหน้าจอขนาดเล็ก

## 🔒 ความปลอดภัย

- ข้อมูลถูกเก็บใน localStorage ของเบราว์เซอร์
- มีระบบล็อกอินควบคุมสิทธิ์
- ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอก (ยกเว้นเมื่อเชื่อมต่อ Google Sheets)

## 📋 ฟังก์ชันหลัก

### การยืมวิทยุ
1. สแกน QR Code หรือเลือกจากตาราง
2. กรอกข้อมูลผู้ยืม (ชื่อ, เบอร์โทร, แผนก)
3. ถ่ายรูปผู้ยืม
4. คลิก "บันทึกการยืม"

### การคืนวิทยุ
1. สลับไปแท็บ "คืนวิทยุ"
2. ค้นหารายการที่ต้องการคืน
3. คลิก "คืน" ในรายการนั้น

### การดูรายงาน
- ดูสถิติในการ์ดสรุป (ว่าง/ยืม/ทั้งหมด)
- ค้นหารายการในตาราง
- Export ข้อมูลเป็น CSV
- พิมพ์เอกสารรายการ

## 🛠️ การปรับแต่ง

### เปลี่ยนจำนวนวิทยุ
แก้ไขค่า `this.TOTAL` ในไฟล์ `app.js`:

```javascript
this.TOTAL = 84; // ปัจจุบันตั้งไว้ที่ 84 ตามจำนวนวิทยุที่มี
```

### เปลี่ยนภาษา
ระบบใช้ภาษาไทยเป็นหลัก สามารถแก้ไขข้อความใน HTML และ JavaScript ได้

### เปลี่ยนสีธีม
แก้ไข CSS variables ในไฟล์ `styles.css`:

```css
:root {
  --accent: #3b82f6;      /* สีหลัก */
  --green: #10b981;       /* สีสำเร็จ */
  --red: #ef4444;         /* สีแจ้งเตือน */
  /* ... */
}
```

## 🐛 การแก้ไขปัญหา

### กล้องไม่ทำงาน
- ตรวจสอบว่าเบราว์เซอร์อนุญาตให้ใช้กล้อง
- ใช้ HTTPS หรือ localhost เท่านั้น
- ลองรีเฟรชหน้าเว็บและขออนุญาตใหม่

### สแกน QR ไม่ได้
- ตรวจสอบว่ากล้องหลังทำงาน
- แน่ใจว่ามีแสงพอสำหรับสแกน
- ลองใช้การเลือกจากตารางแทน

### ข้อมูลหาย
- ข้อมูลถูกเก็บใน localStorage ของเบราว์เซอร์
- ห้ามลบ cache หรือใช้โหมดไม่ระบุตัวตน
- ควรสำรองข้อมูลด้วย Google Sheets
